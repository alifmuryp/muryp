Penjelasan singkat mengenai Shared hosting, Dedicated Hosting , Cloud Hosting, dan VPS

Hello world!

Pernahkah kalian mendengar istilah Shared hosting, Dedicated Hosting , Cloud Hosting, dan VPS? Tau kalian bingung mau pilih yang mana? Oke mari kita bahsa satu per satu.


Pertanyaan yang harus bisa kalian jawab sendiri untuk bisa memilih mana yang terbaik :

  1. Berapa anggaran bulanan Anda?
    • Jika tipis bisa pilih shared hosting
  2. Apakah lalulintas web anda banyak atau biasa saja?
    • Jika tinggi gunakan dedicated, cloud atau jika punya dana tinggi bisa gunakan VPS
  3. Apakah web anda sering terjadi sebuah lonjakan pengunjung?
    • Jika ya, bisa gunakan Clod ataupun VPS
  4. Berapa tinggi kemampuan/skill mengelolah web?
    • Jika anda atau pegawai anda tak mahir atau tak mempunyai tim IT yang mumpuni saya bisa rekomendasikan untuk pilih pakai Shared Hosting, atau Dedicated Hosting. Untuk cloud dan VPS tak direkomendasikan bagi pemula.
  5. Apakah ada persyaratan khusus untuk masalah keamanan?
    • Jika kalian punya tim IT pasti tim IT kalian akan memilihkan Cloud atau VPS jika butuh kemanan yang lebih.
  6. Apakah anda butuh performa tinggi?
    • Jika ya jangan gunakan shared hosting.


Penjelasan


1. Shared Hosting

Dari Q&A diatas kita tahu bahwa kita simpulkan Shared Hosting itu Murah, Mudah, namun keamanannya tak bisa dijagakan dan tak bisa menampung pengunjung yang banyak. Mesakipun begitu bagi kalian yang pemula yang hanya menggunakan untuk blog atau hal lainnya bisa menggunakannya. Dan tak semua shared hosting itu sama. Saya hanya memberikan gambaran mengenai rata-rata shared hosting. Itu semua tergantung penyedia layanannya.


2. Dedicated Hosting

Dan yang kedua ini lebih bagus dari yang pertama. Mulai dari mampu menampung pengunjung yang lebih banyak dan lebih optimal. Anda bisa memilih aplikasi apa aja. Kemanan yang lebih bagus, karena tidak berbagi bandwith. Dengan syarat kalian harus merogoh kocek lebih besar.



3. Apa itu Cloud Hosting?

Cloud cukup mirip dengan VPS namun berbeda dengan VPS. Dimana VPS bergantung dengan server fisik sedangkan Cloud mempunyai beberapa server yang saling berbagi membentuk sebuah cloud. Contoh cloud untuk menyimpan sebuah file yang paling terkenal ialah Google Drive. Untuk yang web kita bisa lihat di penyedia layanan seperti Niagahoster, Exabyte, dan penyedia layanan lainnya.


Selain itu VPS memiliki batasan pada jumlah bandwidth dan transfer data yang dapat disediakannya, sedangkan cloud hosting dapat memanfaatkan sumber daya yang sama berulang kali. Analogi VPS ialah saat kalian saat kalian bermain laptop tanpa chas, baterai laptop pasti nantinya akan habis dan mati. Sedangakan Cloud menggunakan charger agar baterai terus terisi selama daya terus terisi. 


Dan keuntungan terbesar dari Cloud Hosting yaitu kita bisa menambah penyimpanan tanpa migrasi. Jika salah satu server error maka akan dialihkan ke server awan lainnya. Sumber daya sesuai permintaan. Stabilitas server cloud tidak tertandingi di bidang web hosting.


FYI : Jika Perusahaan besar seperti Google dan Amazon menghosting server mereka di sistem cloud, pasti ada keuntungannya.


4. Apa itu VPS? 


Peladen pribadi virtual, (bahasa Inggris: Virtual private server; VPS) adalah teknologi sisi peladen tentang sistem operasi dan perangkat lunak yang memungkinkan sebuah mesin dengan kapasitas besar dibagi ke beberapa mesin virtual. Tiap mesin virtual ini melayani sistem operasi dan perangkat lunak secara mandiri dan dengan konfigurasi yang cepat. Peladen terdedikasi virtual (virtual dedicated server/VDS) juga memiliki arti yang serupa.

(Sumber Wikipedia)

Dengan pengertian yang dikutip dari wikipedia dan pembahasan sebelumnya. Kita dapat menyimpulkan bahwa VPS sebuah server virtual yang bisa dioperasikan dimana saja asalkan ada internet. Bagi kalian yang sudah pernah membuat server mini dengan komputer atau lainnya. Itu mirip, hanya saja kita memindahkan kerumitan pemeliharaan hardware ke penyedia layanan VPS. Kita sudah tak perlu memikirkan bagaimana perawatan dan koneksinya. Semua sudah dihandle penyedia layanan. Bisa dibilang nitip server. Sehingga harga VPS cenderung mahal. Karena kita menyewa part hardware dari sebuah server. 


Dan keuntungan menggunakan VPS antara lain, kita bisa membuat beberapa akun hosting di VPS kita. Tak hanya menjalakan sistem Web hosting saja. Kalian akan mendapat akses root. Seperti cloud kita dapat menambah penyimpanan tanpa migrasi. Kita bisa memilih, mengganti, menghapus sebuah software pada VPS. Kita bisa menyesuaikan panel yang kita inginkan. Kita juga bisa mengatur sendiri kemanan sesuai kemauan kita. Server cenderung stabil. Memiliki sumber daya yang besar. Terdapat IP Dedicated tanpa merogoh kocek lagi. Dan yang terakhir uptime yang lebih tinggi.


Kesimpulan

Pemilahan sendiri tergantung kebutuhan kita. Jika kalian pemula dan ingin belajar bisa gunakan shared hosting yang lebih murah. Jika kalian ingin kecepatan yang stabil, mungkin trafik kalian gede namun belum, bisa gunakan shared hosting atau Cloud ataupun VPS. Jika kalian ingin yang bagus mungkin saran saya gunakan cloud aja. Dan semua itu tergantung kebutuhan masing-masing
